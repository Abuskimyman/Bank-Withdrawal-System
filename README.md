# BANK WITHDRAWAL SYSTEM

A bank withdrawal system where users enter an amount. Raise an exception if the withdrawal exceeds the balance, confirm success in the else block, and log the transaction in finally.

## FEATURES
- User inputs withdrawal amount.
- Checks if withdrawal exceeds available balance.
- Raises a custom exception for insufficient funds.
- Displays success message on valid transaction.
- Logs transaction in the finally block.

## HOW TO RUN IT
- It starts with a default account balance.
- The user is promted to type in the amount they want to withdraw.
- If the amount is greater than the balance, an exception is raised.
- If the withdrawal is successful, the balance is updated.
- A transaction message is displayed in all cases.

## REQUIRED LIBRARIES
- Python 3.x (standard library only)
